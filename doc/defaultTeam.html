    public ArrayList calculFVS(ArrayList points)

    Description

    Point d'entrée de notre algorithme, méthode prenant la liste de points a traier en paramètre et retournant le resultat de l'algorithme Bafna-Berman-Fujito. La liste commence par transformer les points en FVSPoint puis appel la méthode start exécutant l'algorithme Bafna-Berman-Fujito, puis finalement reconvertie les points résultants en FVSPoint et retourne la liste des points à retirer
    param points : liste de points a traiter
    return : liste des points à retirer du graphe apràs traitement de l'algorithme Bafna-Berman-Fujito
    private ArrayList fvsListToPointList(ArrayList fvsPoints)

    -> l'appel à la méthode modifie les poids des élements de la liste comme suit :
    -> les poids de tous les points sont égaux au poid minimal dans la liste, donc tous les points ont un poid égal à 0.
    private void processSemidisjointNotFound(ArrayList points)

    private ArrayList getSemidisjointCycle(FVSPoint parent, FVSPoint current, ArrayList path)

    Description

    Recherche un cycle semi disjoint en debutant le parcours depuis current
    param parent : le parent du noeud courant null au debut
    param current : noeud courant
    param path : liste de noeuds parcourus
    return : la liste contenant le cycle semi disjoint

    Exemple

    fvs1 = FVSPoint((20,70), 1)
    fvs2 = FVSPoint((90,8), 0)
    fvs3 = FVSPoint((70,8), 0)
    fvs4 = FVSPoint((155,80), 3)
    path = []
    points = [fvs1, fvs2, fvs3, fvs4]
    current = fvs1
    parent = null
    -> la méthode retourne la liste [fvs1, fvs2, fvs3]
