    private boolean isMember(ArrayList points, Point p)

    Description

    Vérifie si le point p se trouve dans la liste points, retourne true si c'est la cas et false sinon
    param points : liste de points
    param p : point recherche
    return 'true' si p appartient à la liste points, 'false' sinon

    Exemple

    On considère:
    points [(1,6), (5,9), (2,5)]
    p (5,9)
    -> La méthode retourne true
    public boolean isValide(ArrayList origPoints, ArrayList fvs)

    Description

    Vérifie si en retirant les points de la liste fvs de la liste origPoints, le graphe resultant reste valide
    param origPoints : graphe original
    param fvs : liste des points à retirer
    return 'true' si le graphe resultant est valide, 'false' sinon

    Exemple

    Point fvs1 = new Point(20, 70)
    Point fvs2 = new Point(90, 8)
    Point fvs3 = new Point(70, 8)
    Point fvs4 = new Point(155, 80)
    origPoints = [fvs1, fvs2, fvs3, fvs4]
    fvs = [fvs4]
    -> L'appel a la methode retourne false, car fvs1, fvs2 et fvs3 forment un cycle
    -> Si on remplace donc la parametre origPoints par: origPoints = [fvs1, fvs2, fvs4] la méthode retourne 'true'. Car après suppresion de fvs4 il ne reste plus que fvs1 et son voisin fvs2. Ce qui forme un graphe valide.
    private ArrayList neighbor(Point p, ArrayList vertices)

    Description

    Retourne la liste des voisins d'un point p dans la liste de points verticles, un voisin est un point qui est à une distance < 100
    param p : un point
    param vertices : une liste de points
    return : la liste de voisins du point p

    Exemple

    On considère:
    p (2,5)
    vertices [(50,70), (10,8), (200,8), (130,80)]
    -> La méthode retourne [(50,70), (10,8)]
