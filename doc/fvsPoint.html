  • Classe definissant un FVSPoint. Un FVSPoint contient un noeud root, un poid w et un boolean isUsed servant dans la detection de cycle comme marqueur de passage


    • Contructeur

    public FVSPoint(Point root)

    Description

    Constructeur prenant un point en parametre, le poid w est initialisé à 1. param root : valeur affecte au noeud root de l'objet
    public FVSPoint(Point root, double w)

    Description

    Constructeur prenant un point et un poid en parametre, et initialisant les champs concernès.
    param root : valeur affecte au noeud root de l'objet
    param w : poid du noeud

    Methodes
    public Point getRoot()

    Description

    Retourne le noeud root de l'objet
    return : root

    Exemple

    fvs = FVSPoint((2,5))
    -> l'appel de la méthode sur fvs retourne (2,5)
    public double w()

    Description

    Retourne le poid w du noeud
    return : w

    Exemple

    fvs = FVSPoint((2,5), 3)
    -> l'appel de la méthode sur fvs retourne 3
    public void setW(double w)

    Description

    Affecte une nouvelle valeur au poid w du noeud
    param w : nouveau poid

    Exemple

    fvs = FVSPoint((2,5), 3)
    fvs.setW(5)
    w = fvs.w()
    -> l'affichage de w donne '5'
    public boolean getIsUsed()

    Description

    Vérifie si le noeud courant a été marque
    return : isUsed

    Exemple

    fvs = FVSPoint((2,5))
    -> l'appel à la méthode sur fvs retourne false
    public void setIsUsed(boolean val)

    Description

    Modifie l'attribut isUsed pour marquer ou démarquer un noeudw
    param val : nouvelle valeur de isUsed

    Exemple

    fvs = FVSPoint((2,5))
    fvs.setIsUsed(true)
    -> l'appel à la méthode getIsUsed sur fvs retourne true
